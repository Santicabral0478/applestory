Macintosh 128K

it offers a graphical user interface, an advanced processor and improved RAM memory that enables multitasking. It comes with a mouse and a keyboard. These features make the Macintosh 128K easier to use and more accessible to non-technical users.

Operating System: MacOS 1.0 operating system.

Processor: Motorola 68000 8 MHz.

RAM memory: 128 KB of RAM memory.

Storage: 400KB that could read and write to 3.5-inch drives.
