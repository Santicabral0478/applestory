Apple Lisa

one of the first personal computers to use a graphical user interface and mouse, and offered a variety of features, including a start screen with application icons, a Lisa OS operating system, an application folder, pull-down menus, windows, a hierarchical file system and a built-in printer.

Mouse: single-button mouse that connects to the central unit.

Operating system: Lisa OS operating system, offering a graphical user interface and a number of built-in applications,

Folder: On the home screen, there is a folder where all the applications installed on the Apple Lisa are located. To open an app, just click on its icon.

Menu: The dropdown menus contain a variety of options for customizing and controlling the application.
