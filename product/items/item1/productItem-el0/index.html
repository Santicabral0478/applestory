Apple I

the first computer from apple computer. it must be connected to a power supply, a monitor and an ASCII keyboard in order to be used. Its most common programming language is BASIC.

microprocessor: 8-bit MOS 6502

Memory: 4KB RAM

programming language: BASIC

connectivity: ports for ASCII keyboard, monitor and cassette to store data
